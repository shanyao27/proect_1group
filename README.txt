Для запуска проекта необходимо:
1) Скачать его с сайта GitHub или добавить локальную копию с помощью git clone https://github.com/shanyao27/proect_1group
2) Написать в консоли: cd /путь к src файлу проекта
3) Написать в консоли: npm start
4) При появлении критических уведомлений об ошибках установить библиотеки, которые там фигурируют
5) Оценить качество и работоспособность прототипа
